## 魔剑

由景天的前世姜国太子龙阳所铸，因姜国公主龙葵的室女之血而成。

## 模块

- `sunchaser-mojian-base`: 提供基础能力。
- `sunchaser-mojian-util`: 提供常用工具。
- `sunchaser-mojian-web`: 提供 `web` 开发能力。
- `sunchaser-mojian-log`: 提供日志切面能力。
